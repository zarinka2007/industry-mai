# 🏭 Наследие индустрии: Умный подбор локации и дизайна производства

> **«Проектировать промышленные объекты нового поколения, которые органично вписываются в культурный ландшафт региона, уважают его историю и становятся точкой притяжения для местных сообществ.»**

## 📌 О проекте
Веб-приложение (MVP) для инвесторов и администраций, которое по 10 параметрам производства автоматически:
- Выдаёт **ТОП-3 региона** с промышленными площадками и полной аналитикой.
- Формирует **концепт-борд** (цвета, материалы, архитектурный стиль) на основе культурного кода региона.
- Генерирует **интерактивную 3D-визуализацию** производственного участка.
- Подготавливает **аналитическую справку** (соц. паспорт, экономика, сети, логистика, смета).
- Создаёт **презентацию для администрации** (6 слайдов) в формате HTML.

**Тип производства:** Сэндвич-панели (с утеплителем ППУ / минвата / ППС).

## ⚙️ Как это работает (Алгоритм)
1. **Ввод данных:** Инвестор заполняет 10 полей (объем выпуска, бюджет, логистика, соц. приоритеты).
2. **Расчет площадей:** Система считает площади цеха, склада, жилья и соц. объектов по нормативам ТЗ (п.6).
3. **Ранжирование регионов:** Каждый из 8 регионов оценивается по 5 весовым категориям:
   - 🚚 **Логистика (25%):** Близость к поставщикам стали и утеплителя.
   - 💰 **Экономика (25%):** Наличие ОЭЗ/ТОР, налоговые льготы, энерготарифы.
   - 🔌 **Инфраструктура (20%):** Доступность газа, свободные мощности э/э, Ж/Д ветки.
   - 👥 **Социум (15%):** Индекс качества среды, доступность детсадов.
   - 🌿 **Экология (15%):** Класс загрязнения воздуха (ИЗА).
4. **Генерация outputs:** Для ТОП-3 регионов формируются карта, концепт-борд, 3D-модель и смета.

## 🚀 Инструкция по запуску (Локально)

   ```bash
   git clone <ваш-url-репозитория>
   cd heritage-industry
2. Создайте виртуальное окружение и установите зависимости:
```bash
python -m venv venv
source venv/bin/activate   # Для Windows: venv\Scripts\activate
pip install -r requirements.txt
```
3. Запустите приложение:
````
streamlit run app.py
````
4. Откройте в браузере: http://localhost:8501
